Handy Hotspot NYC

Handy Hotspot NYC is a website created by Gilvir Gill and Henry Zheng. Written in Python, it takes the location given by the user and compares distances between it and a database of hotspots throughout NYC. Once it finds the closest one by using Google Maps Geocode API to get coordinates and then using math to find the distance between 2 given coordinate points, it will then give the user directions to the closest location.

Usage

Enter an address into the text field.
The computer will take your address, and it will find the closest Wi-Fi hotspot.
Select type of hotspot and type of location preferred.

Pro Tips:

  1. Make sure to enter an address in the correct format.
  2. Select at least one type of hotspot and one type of location
  3. Keep in mind the database only tracks hotspots in New York City
  4. Do not enter a semi-colon or other weird characters in your address
